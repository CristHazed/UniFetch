Install Guide

Before You Open The App

Android

Download the APK, allow installation from your browser or file manager, then open UniFetch.

Windows

Download the ZIP, extract the folder, then double-click Install-UniFetch.bat. A UniFetch shortcut will be added to the Desktop.

Accounts

Create an account with your student details. Driver access opens after verification and background checking.
