UniFetch Desktop Installer

How to install:
1. Extract this ZIP first.
2. Double-click Install-UniFetch.bat.
3. After installation, open UniFetch from the Desktop shortcut.

Requirement:
Java 17 or newer must be installed on the computer.

How to uninstall:
Open this folder after installation:
%LOCALAPPDATA%\UniFetch
Then run Uninstall-UniFetch.bat.
